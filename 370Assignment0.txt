2. C++ (cpp --version) and Python (python --version). A high level language is either compiled or interpreted. In a compiled language such as C or C++, a compiler translates a high leve language into machine code that a computer can understand. To do this, a compiler performs lexical analysis to break source code into tokens such as keywords, identifiers, operators, and punctuation, then syntax analysis where the compiler parses source code, checking for syntax errors and generating a syntax tree to represent the code's structure. Next, the compiler performs semantic analysis to find the meaning of the code and check for logical code errors. Optionally, the compiler may perform optimization to improve code performance, and finally, code generation where the compiler generates assembly code which is translated to machine code via an assembler that is readable by the computer. Interpreters for interpreted languages follow a similar process, however the code is translated to machine code line by line.

8. Photolithography is a process used to place structures on the surface of a polymer-based photoresist. Using light, a specific geometric pattern is transfered to the photoresist by inducing chemical changes that allow the photoresist to be removed later in the process. This transfer is done either through direct laser writing or to exposure of UV light or X-rays through a photomask. UV lithography can generate patterns down to 50-100 nm. The Intel(R) Xeon(R) Gold 6230 processor transistors were manufactured using the 14 nm node.
